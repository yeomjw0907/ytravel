# Ytravel 화면 구조 및 와이어프레임 초안

## 1. 전체 구조

MVP 기준 페이지 구조

- `/`
  - 홈 및 검색 시작 페이지
- `/search`
  - 검색 결과 페이지
- `/hotel/[slug]`
  - 호텔 상세 비교 페이지

## 2. 홈 화면

목적

- 사용자가 Ytravel의 기능을 바로 이해하고 검색을 시작하게 만든다.

섹션 구성

- 헤더
  - 로고: Ytravel
  - 내비게이션: 서비스 소개, BRG란?, 검색 시작
- 히어로
  - 카피: `호텔 공홈보다 싼 요금, Ytravel이 먼저 찾습니다`
  - 서브카피: 공홈, Trip.com, Agoda, Booking.com을 BRG 관점으로 비교
  - 검색 폼
- 설명 섹션
  - BRG 동작 방식 3단계
- 신뢰 섹션
  - 어떤 조건을 비교하는지 요약
- 푸터
  - 면책 문구
  - 문의 링크

와이어프레임 초안

```text
+------------------------------------------------------+
| Ytravel                         BRG란?   검색시작 |
+------------------------------------------------------+
| 호텔 공홈보다 싼 요금, Ytravel이 먼저 찾습니다      |
| 공홈/OTA 가격을 같은 조건으로 비교하고 BRG 가능성을 표시 |
| [호텔명         ] [체크인] [체크아웃] [검색하기]         |
+------------------------------------------------------+
| BRG는 이렇게 찾습니다                                  |
| 1. 공홈 확인  2. OTA 비교  3. 조건 검토                |
+------------------------------------------------------+
```

## 3. 검색 결과 화면

목적

- 공식가와 OTA 가격을 빠르게 비교하고 BRG 후보를 판단하게 만든다.

섹션 구성

- 상단 검색 요약 바
  - 호텔명
  - 날짜
  - 인원
  - 검색 수정 버튼
- 호텔 요약
  - 호텔명
  - 위치
  - 공식 사이트 링크
- BRG 요약 박스
  - 공식가
  - OTA 최저가
  - 차액
  - BRG 가능성 상태
  - 예상 BRG가
- 공급처 비교 리스트
  - 카드 또는 테이블 형태
  - 가격 기준 정렬
- 조건 비교 요약
  - 취소 가능
  - 조식 포함
  - 세금 포함
  - 결제 방식

와이어프레임 초안

```text
+------------------------------------------------------+
| Grand Hyatt Seoul | 2026-05-10 ~ 2026-05-12 | 2명     |
+------------------------------------------------------+
| BRG 가능성 높음                                       |
| 공식가 320,000원 | OTA 최저가 268,000원 | 예상 BRG가 |
+------------------------------------------------------+
| Official  | 320,000 | 무료취소 | 조식없음 | 보기       |
| Trip.com  | 268,000 | 무료취소 | 조식없음 | 보기       |
| Agoda     | 279,000 | 무료취소 | 조식없음 | 보기       |
| Booking   | 291,000 | 무료취소 | 조식없음 | 보기       |
+------------------------------------------------------+
| 조건 비교: 객실명 유사 / 취소정책 일치 / 세금 포함 동일 |
+------------------------------------------------------+
```

## 4. 호텔 상세 페이지

목적

- 실제 BRG 판단을 위한 세부 근거를 확인하게 만든다.

섹션 구성

- 호텔 헤더
- 가격 타임라인 또는 가격 카드
- 객실 조건 비교 표
- 공급처 링크
- BRG 진행 가이드
- 면책 및 주의사항

## 5. 컴포넌트 후보

- SearchBar
- HotelHeader
- BrgSummaryCard
- ProviderRateCard
- ProviderRateTable
- ConditionBadge
- MatchConfidencePill
- BrgGuidePanel

## 6. UX 원칙

- 최저가와 공식가를 시각적으로 분리한다.
- 단순 숫자 비교보다 조건 차이를 먼저 보이게 한다.
- BRG 가능성을 색상과 문구로 즉시 이해 가능하게 한다.
- 모바일에서는 카드형, 데스크탑에서는 비교표 중심으로 노출한다.
